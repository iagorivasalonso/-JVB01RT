Debes desarrollar el backend lógico para una aplicación de gestión de clientes. La aplicación debe permitir a los usuarios, a través de un menú en la consola, las siguientes operaciones:


Agregar un nuevo cliente: Los usuarios deben poder ingresar información sobre un nuevo cliente, incluyendo nombre, apellidos, sexo, ciudad, fecha de nacimiento, teléfono, correo electrónico.

Listar todos los clientes: La aplicación debe permitir la visualización de la lista de todos los clientes en la base de datos.

Actualizar información de un cliente: Los usuarios deben poder modificar la información de un cliente existente.

Eliminar un cliente: La aplicación debe permitir eliminar un cliente de la base de datos.

Buscar cliente por ciudad: Los usuarios deben poder buscar clientes por ciudad y ver una lista de los clientes que son de esta ciudad.



Requisitos técnicos:


Utiliza Java para el desarrollo de la aplicación.
Debes utilizar estructuras de control para la validación de datos de entrada, como asegurarte de que los campos obligatorios no estén vacíos (esto es obligatorio) y que los valores sean válidos (esto es opcional).
La aplicación debe utilizar JPA para acceder a la base de datos. Asegúrate de configurar una fuente de datos y una entidad "Cliente" con las propiedades necesarias.
Utiliza colecciones para gestionar los datos antes de interactuar con la base de datos.
